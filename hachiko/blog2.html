เทคโนโลยีสำหรับรองเท้าของเรา

ด้วยความเชี่ยวชาญกว่า 40 ปีทั้งการวิจัยและการพัฒนาที่มีเราอยู่เบื้องหลัง เราได้ทำทุกอย่างที่ทำได้เพื่อทำให้รองเท้านิรภัยของเราสบายมากขึ้น เราได้พัฒนาเทคโนโลยีที่พิเศษเฉพาะ 3 รายการโดยมีเป้าหมายที่จะมอบความสะดวกสบายมากยิ่งขึ้น

สิ่งที่เป็นแรงบันดาลใจให้เรา:รูปร่างตามธรรมชาติของเท้า ที่มีส่วนโค้ง รูปร่างทั้งหมดนี้มีส่วนประกอบของกระดูก ข้อต่อ เส้นเอ็น และกล้ามเนื้อของเท้าเข้าด้วยกันอย่างชาญฉลาด

เหตุใด?เพราะอะไรทำให้ความโค้งและความยืดหยุ่นที่เปลี่ยนแปลงไป ? การปรับส่วนโค้งให้เข้ากับสภาพพื้นดินได้ มันทำหน้าที่เป็นตัวกันลื่นซึ่งเป็นสิ่งจำเป็นต่อความยืดหยุ่น

สัญลักษณ์แห่งการกระทำ ความสามารถในการปรับตัวและก้าวไปข้างหน้านี้แสดงเป็นรูปทรงกลมของเทคโนโลยีพิเศษ เฉพาะของเรา 3 รายการ

VPS* SYSTEM (SUSPENDED ARCH SUPPORT)

รูปทรงตามหลักทางกายภาพของรองเท้าช่วยรักษาโครงสร้างของเท้าได้อย่างดี โดยจะยึดตามส่วนโค้งของเท้า และการรองรับภายในของพื้นรองเท้านี้จะสร้างความสบาย โดยเฉพาะในกรณีของการเดินหรือการยืนในเวลานานๆ โดยไม่มีการเคลื่อนไหว

DRS (DELETE RESONANCE SYSTEM) / DRS 2

เทคโนโลยี DRS ของเราอยู่ที่ส้นเท้าและทำหน้าที่เป็นตัวกันลื่น DRS คือช่องใต้ส้นเท้าที่มีรูปร่างคล้ายร่องเสี้ยวครึ่งวงกลม เมื่อส้นเท้าสัมผัสกับสภาพพื้นผิว ช่องนี้จะตกลงเข้าด้านในและเคลื่อนไปตามส้นเท้าเพื่อปรับให้เข้ากับสภาพพื้นผิว ทำให้การเคลื่อนไหวไปข้างหน้าของเท้าเป็นไปอย่างดียิ่งขึ้น คุณจะพบกับ DRS 2 ได้บนพื้นรองเท้าแบบ Racing sport ของเรา

VPS* SYSTEM (SUSPENDED ARCH SUPPORT)

เทคโนโลยี PARADE AIR SYSTEM ของเรายังได้รับแรงบันดาลใจจากรูปทรงออร์แกนิกและความกลมของส่วนโค้งของเท้าอีกด้วย เทคโนโลยี PARADE AIR SYSTEM คือฟองอากาศที่ห่อหุ้มอยู่ที่พื้นรองเท้าชั้นนอกของรองเท้า เทคโนโลยีเนี้ช่วยเพิ่มความยืดหยุ่นในขณะที่เดินไปข้างหน้า

โล่ที่1 : โล่แห่งการป้องกันรอยขีดข่วนของเรา

วัสดุ SHIELD ONE พิเศษเฉพาะของเราเป็นเกราะป้องกันรอยขีดข่วนอย่างแท้จริง ระบายอากาศได้ดีด้วยดีไซน์ Parade Shield ช่วยป้องกันการสึกหรอ สูตรผ้า + TPU ตรงตามข้อกำหนดด้านความทนทานต่อการเสียดสีที่กำหนดโดยมาตรฐานถุงมือ EN388 (ประสิทธิภาพระดับ 4/4) วัสดุที่มีความทนทานสูงนี้มีอยู่ในรองเท้า S1P ของเราในกลุ่มผลิตภัณฑ์ RACING: Ride and Racer

FLEX SYSTEM

เทคโนโลยี FLEX SYSTEM ของเราเหมาะสำหรับการทำงานที่ต้องการความยืดหยุ่นและแข็งแรงมากที่สุด เทคโนโลยีนี้เหมาะสำหรับงานก่อสร้างและพื้นผิวที่ไม่เรียบ ด้วยการออกแบบพิเศษของพื้นรองเท้าที่มีร่องลึกลดพื้นที่ที่มีแรงกดบนพื้นรองเท้า จึงมอบความยืดหยุ่นและความสบายในการเดินได้ เมื่อเท้าเคลื่อนไหวรองเท้าจะเคลื่อนไหวไปพร้อมกับเท้าและร่องลึกแนวนอนกับหลุมที่อยู่ใต้กระดูกเท้าช่วงตรงเท้าที่เท้าโค้งได้โดยธรรมชาติและเมื่อเคลื่อนเท้าไปข้างหน้าดินทรายที่รองเท้าจะหลุดออกมาเอง คุณสามารถพบเทคโนโลยี FLEX SYSTEM ของเราในรองเท้าทุกรุ่นที่มีพื้นรองเท้า NEBRASKA และ NEVADA ของเรา

BE FLEX 2.0

เทคโนโลยี BE FLEX 2.0 ออกแบบมาเพื่องานที่ต้องการความยืดหยุ่นและความสะดวกสบายในการเดิน เช่น งานดานโลจิสติกส์ เป็นต้น ส่วนด้านข้างของพื้นรองเท้าถูกยกสูงขึ้นที่ส่วนหน้าของเท้าเพื่อให้ง่ายต่อความยืดหยุ่น ส่วนด้านหน้ามีการออกแบบพิเศษด้วยร่องตามขวาง 3 ช่องเพื่อการเคลื่อนไหวและความยืดหยุ่นสูงสุดสำหรับส่วนหน้าของเท้า

คุณจะพบกับเทคโนโลยี BE FLEX 2.0 ได้ในรองเท้า RACING ของเรา
